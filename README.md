# Ad Composer (Observer)

A simple, powerful tool for analyzing competitor ads and generating marketing strategies.

## Deployment Setup (Cloudflare Pages)

This project is deployed on Cloudflare Pages as a static site.

- **Build Command:** `(empty)` (No build command needed)
- **Output Directory:** `public` (Use the root `public` folder directly)
- **Node Version:** 18+ (Recommended)

## Local Development

To run locally:

```bash
python3 -m http.server 8000 -d public
```

Open `http://localhost:8000` in your browser.
